# EventLogDatasetDurationExtractor
The code here presented is able to extract average duration and max duration from event logs.


* Extracting the average duration of all events.
* Extracting the maximal duration of all events.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 

### Prerequisites

To execute this code you just need to install the requirements in your system, and create an environment using the *requirements.txt* specification provided in the repository.

Once created the environment, you can perform the task running the main.py module:
